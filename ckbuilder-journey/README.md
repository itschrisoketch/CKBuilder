# CKBuilder Journey

My learning log for the Nervos **CKBuilder** track (Community Keeps Building).

This repository tracks, week by week, what I study and build while getting up to speed on
CKB — the Cell Model, CKB-VM, on-chain scripts, and dApp development with the CCC SDK.

## Layout

| Folder | What's in it |
| --- | --- |
| [`notes/`](./notes) | Numbered learning notes as I work through the docs |
| [`experiments/`](./experiments) | Runnable code: offCKB projects, CCC examples, script prototypes |
| [`reports/`](./reports) | Weekly dev-log reports (one per week) |
| [`articles/`](./articles) | Longer-form write-ups |
| [`screenshots/`](./screenshots) | Evidence of progress, organised by week |

## Local setup

- **offCKB** devnet (`offckb node`) — RPC proxy at `http://127.0.0.1:28114`
- **pnpm** for all JS/TS tooling
- **CCC SDK** (`@ckb-ccc/*`) for dApp code

See the repo-root [`CLAUDE.md`](../CLAUDE.md) for working conventions.
